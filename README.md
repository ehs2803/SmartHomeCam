# SmartHomeCam (졸업작품)

## 작품 주제

라즈베리파이와 인공지능을 이용한 스마트 홈캠 웹 서비스

## 주제 선정 배경
1인 가구의 증가와 반려가구 1500만 시대에 따라 외출 중에  카메라를 통해 집안상황 확인, 방범 서비스, 화재감지, 반려동물을 확인을 할 수 있다.
